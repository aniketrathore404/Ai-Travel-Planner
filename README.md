# AI Travel Planner ✈️

A full‑stack project that helps users plan trips with AI‑powered recommendations.  
Built with Node.js, Express, and modern frontend components.

---

## 🚀 Features
- AI‑driven travel suggestions
- Organized folder structure (`services/`, `components/`)
- REST API backend with Express
- CORS enabled for cross‑origin requests
- Easy local setup and deployment

---
